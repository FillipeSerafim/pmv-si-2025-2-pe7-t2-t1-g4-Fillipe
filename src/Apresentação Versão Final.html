Algoritmo de Recomendação de Vendas

Do EDA (Análise Exploratória) à Modelagem Preditiva com CatBoost

Em um mercado de varejo cada vez mais competitivo, a capacidade de sugerir produtos relevantes aos clientes é um diferencial estratégico. Este projeto conceitual visa desenvolver um sistema de recomendação baseado no **Superstore Sales Dataset**, focando em aprimorar a experiência de compra e impulsionar o crescimento do negócio.

Próximo: O Problema Central

O Problema Central e a Justificativa

O Desafio

  • **Sobrecarga de Informação:** Clientes lutam para encontrar produtos relevantes.
  • **Perda de Oportunidades:** Ausência de vendas cruzadas e adicionais.
  • **Experiência Subótima:** Frustração e baixo engajamento com a loja.

A Solução e Seus Objetivos

A solução é um sistema de recomendação que responde à questão: *Como usar dados históricos para sugerir produtos a clientes individuais e a clientes com perfis semelhantes?*

1. Explorar Padrões de Compra 2. Identificar Similaridades (Filtragem Colaborativa) 3. Propor Métricas de Avaliação (Precisão, F1-Score)

Análise Exploratória de Dados (EDA)

A etapa inicial é a **Análise Exploratória de Dados (EDA)** no dataset Superstore. Nosso objetivo é investigar a estrutura, padrões e relações entre variáveis para preparar o *dataset* para a modelagem.

Variáveis Chave

Customer ID, Product ID, Category, Sales, Quantity, Order Date.

Pré-Processamento

Limpeza de *outliers* e tratamento de variáveis categóricas.

Foco

Compreender a sazonalidade e produtos comprados juntos (padrões de consumo).

Modelagem Preditiva: A Escolha do CatBoost

Por que CatBoost?

O CatBoost (Categorical Boosting) foi o modelo que se mostrou **superior** para esta tarefa de classificação multiclasse nominal, superando alternativas como o LightGBM e o XGBoost (na abordagem de regressão ordinal).

  • **Performance Superior:** Demonstrou o melhor equilíbrio entre **Acurácia Nominal** (acertos exatos da classe) e **Erro Ordinal (MAE)**.
  • **Tratamento de Categóricas:** Possui tratamento nativo e otimizado para variáveis categóricas, o que é crucial em um dataset de varejo (e.g., *Product ID*, *Category*).
  • **Viés da Regressão Corrigido:** A abordagem de classificação multiclasse (CatBoost) evitou o viés da regressão ordinal com arredondamento, que falhou em capturar classes de demanda de cauda.

~ 0.70

F1-Score Macro Estimado (Melhor Modelo)

"O CatBoost é inequivocamente o modelo superior. Ele fornece o melhor equilíbrio entre Acurácia nominal (acertos exatos) e erro ordinal (MAE), além de ser robusto na predição de todas as classes de demanda."

Conclusão e Impactos do Projeto

Valor Gerado

Este projeto estabeleceu uma base robusta para um sistema de recomendação de vendas, demonstrando como técnicas avançadas de Machine Learning podem transformar dados históricos em insights acionáveis. O modelo CatBoost, com seu desempenho superior no contexto de dados categóricos do varejo, se mostrou uma solução viável para personalização em escala.

Resultados Alcançados

  • Framework completo de EDA para dados de varejo
  • Pipeline de pré-processamento otimizado para dados categóricos
  • Modelo CatBoost com F1-Score de ~0.70 e robustez nas previsões
  • Arquitetura escalável para implementação em sistemas reais
